# Learner

(не) Активно допиливается каждый день!

<p>Сейчас активно пилю Rest API для этой беды. Возможно это первый открытый крупный пет-проект.</p>

<p>Приложение на Xamarin.Forms, главная идея в создании собственного словаря, который поможет держать под рукой самые нужные слова и/или фразы.
    Для удобства сделан поиск, сортировка, озвучивание слова через text-to-speach. Отдельно стоит выделить создание коллекций, которые помогут каталогизировать слова. 
<br/> Для разнообразия есть мини-игра. Пока подбор слов для квиза идёт из общего списка слов, но в будущем хочу сделать выбор набора слов для повторения.</p>

<h3>Роадмап</h3>
<p>• Привязка словаря к аккаунту</p>
<p>• Локализация</p>
<p>• Кастомное оформление</p>
<p>• Web-версия</p>
<br/>

<h3>Known issue</h3>
<p>• При активной тёмной теме оформления системы, шрифты могут поменять цвет, что сделает их не читаемыми</p>
<p>• Некоторые вещи в состоянии прототипа и реализованы костылями, сортировка и языки будут переделаны</p>
<p>• Несколько неудачных миграций сломали некоторые моменты, многое было пофикшено, но не всё, отдельные куски кода могут отражать старые коммиты</p>
